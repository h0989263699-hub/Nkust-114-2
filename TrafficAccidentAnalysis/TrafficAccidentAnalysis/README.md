# 台灣 A1 類交通事故數據分析

這是一個使用 C# 與 .NET 開發的數據分析工具，專門處理台灣警政署提供的 A1 類重大交通事故開放資料。

## 核心功能
- **JSON 解析**：自動處理與對應複雜的交通數據格式。
- **肇因統計**：分析並排列出最常見的事故主因。
- **車種特徵**：統計涉及事故的高風險車輛類型。

## 數據來源
- 來源：內政部警政署 (NPA) 開放資料。
- 格式：JSON。

## 如何執行
1. 使用 Visual Studio 2022 開啟專案。
2. 確保 `NPA_TMA1_JSON.json` 位於輸出目錄。
3. 直接執行 (F5) 即可看到分析結果。
